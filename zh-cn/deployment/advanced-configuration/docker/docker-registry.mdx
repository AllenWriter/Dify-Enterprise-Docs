## 在 Helm Chart 值中找到 Docker Registry 配置
### 显示 Helm Chart 值

```bash
helm show values dify/dify
````

以下 Helm chart 值中 Docker 仓库配置的示例。您可以在 Helm chart 值中设置镜像仓库和标签。

````yaml
api:
  image:
    repository: langgenius/dify-api
    tag: "x.x.x"
  ... ...
````


## 如何将镜像同步到您的私有仓库
您可以使用 `sync.sh` 脚本将镜像同步到您的私有仓库。Dify 镜像支持 ARM64 和 AMD64 架构。

> 点击获取[脚本文件](./sync.sh)。

### 设置 shell 变量
编辑 `sync.sh` 文件并设置以下变量：
- `REGISTRY_URL`：您的私有仓库的 URL
- `NAMESPACE`：您的私有仓库的命名空间

### 同步镜像
在同步镜像之前，您需要先登录到您的私有仓库。

````bash
# ./sync.sh <镜像> <版本>
./sync.sh dify-api x.x.x
````

---
title: Docker Image Pull Secret
---

Dify Enterprise uses a private Docker registry to store Docker images. To pull images from the private registry, you need to create a Docker image pull secret.

- dify-enterprise
- dify-enterprise-frontend

```bash
kubectl create secret docker-registry dify-artifacts-token \
    --docker-server=artifacts.langgenius.ai \
    --docker-username=xxx \
    --docker-password=xxx
```

---
title: Dockerイメージプルシークレット
---

Difyエンタプライス版は、Dockerイメージを保存するためにプライベートDockerレジストリを使用します。プライベートレジストリからイメージをプルするには、Dockerイメージプルシークレットを作成する必要があります。

- dify-enterprise
- dify-enterprise-frontend

```bash
kubectl create secret docker-registry dify-artifacts-token \
    --docker-server=artifacts.langgenius.ai \
    --docker-username=xxx \
    --docker-password=xxx
```

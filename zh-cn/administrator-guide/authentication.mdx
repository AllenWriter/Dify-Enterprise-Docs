---
title: 认证方式
---

Dify 企业版提供三种身份认证方式：邮箱与密码、邮箱与验证码、企业级单点登录（SSO）。系统管理员可根据业务需求，灵活配置 Dify 企业版平台的身份认证方式，统一管理组织成员对 Dify 工作空间与 Web 应用的访问权限。

下表是不同身份认证方式所能应用的范围：

| 认证方式 | 工作空间 | Web 应用 |
|----------|----------|----------|
| 邮箱与密码 | ✓ | X |
| 邮箱与验证码 | ✓ | X |
| 企业级单点登录（SSO） | ✓ | ✓ |

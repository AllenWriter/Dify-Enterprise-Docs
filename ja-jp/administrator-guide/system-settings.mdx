---
title: システム設定
version: '日本語'
---

システム管理人は、Settings ページで現在のエンタープライズ版のシステム管理人ユーザーを管理できます。以下の権限を持っています：
- エンタープライズ版管理コンソールへのログイン
- チームメンバーの追加 / 削除 / 無効化
- Workspace にオーナーを追加
- Workspace に新しいメンバーを追加し、権限を割り当てる

## システム管理人の追加

エンタープライズ版コンソールの作成者はデフォルトで最初のシステム管理人です。任意のシステム管理人は他の人をシステム管理人として招待できます。Settings ページの右上の青い Add ボタンをクリックし、ユーザーの名前とメールアドレスを入力して追加を完了します。
![](/ja-jp/img/cb2bad848e40760140e7aa6e25eb2a42.png)

あるシステム管理人のメールアドレスが "Members" リストに含まれていない場合、その人は Workspace を作成および使用する権限を持っていません。システム管理人が Workspace を作成または参加できるようにするには、[Workspace にメンバーを追加する](/administrator-guide/members-management)を参照してください。

## システム管理人情報の変更

特定のシステム管理人を選択し、右側のメニューボタンをクリックして Edit を選択し、表示されるタブページでシステム管理人の名前またはメールアドレスを変更します。

![](/ja-jp/img/174d265a01cca44c60ae34dc9c0ff4a5.png)

## システム管理人のパスワードリセット

メンバーがパスワードを紛失した場合、システム管理人が手動でパスワードリセットメールを送信する必要があります。メンバーの右側のメニューボタンをクリックして Reset Password を選択します。

![](/ja-jp/img/aa143894091c9e4990c5993163df637a.png)

## システム管理人の無効化

特定のシステム管理人を選択し、右側のメニューボタンをクリックして Edit を選択し、メンバーの Status を banned に切り替えます。無効化された管理人はエンタープライズ版コンソールにアクセスできません。

![](/ja-jp/img/3ee32101ff701db88b8b16e7a6db4587.png)

## 管理人の削除

⚠️ 危険な操作のため、慎重に処理してください
システム管理人の右側のメニューボタンをクリックして Delete を選択します。

## 管理人の検索

検索ボックスにメールアドレスを入力してシステム管理人を検索できます。

![](/ja-jp/img/487f328e311a0242d7216a4a8ce15ba7.png)

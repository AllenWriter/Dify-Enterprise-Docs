## Helm Chartの値中からベクトルデータベースの設定を見つける

### Helm Chartの値を表示

```bash
helm show values dify/dify
```

## Helm Chartの値中からベクトルデータベースの設定を見つける
これはHelm Chartの値におけるベクトルデータベースの設定の例です。

````yaml
ベクトルデータベース:
  useExternal: true
  externalType: "qdrant"
  externalQdrant:
    endpoint: "http://your-qdrant-cluster-url.qdrant.tech/"
    apiKey: "#REPLACE_ME#"
  ... ...
````

## `ベクトルデータベース.externalType` フィールドのオプション
### 企業サポートされているタイプ
- qdrant
- weaviate
- elasticsearch

### コミュニティサポートされているタイプ
- milvus
- relyt
- pgvecto_rs
- tcvectordb
- opensearch

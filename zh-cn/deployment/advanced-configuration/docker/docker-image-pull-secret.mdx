---
title: Docker 镜像拉取密钥
---

Dify Enterprise 使用私有 Docker 仓库来存储 Docker 镜像。要从私有仓库拉取镜像，您需要创建一个 Docker 镜像拉取密钥。

- dify-enterprise
- dify-enterprise-frontend

```bash
kubectl create secret docker-registry dify-artifacts-token \
    --docker-server=artifacts.langgenius.ai \
    --docker-username=xxx \
    --docker-password=xxx
````
